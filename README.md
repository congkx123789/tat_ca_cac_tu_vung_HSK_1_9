# tat_ca_cac_tu_vung_HSK_1_9

This repository contains refined HSK (1 to 9) vocabulary lists enhanced with Sino-Vietnamese meanings, Pinyin, and Vietnamese translations.

## Files Included

- **`hsk_1_to_9_practice_cleaned.csv`**
  The primary vocabulary list containing multi-character words with their corresponding Pinyin, Sino-Vietnamese readings, and Vietnamese meanings.

- **`hsk_1_to_9_chars_practice.csv`**
  A comprehensive list of individual Chinese characters from the HSK curriculum. It includes Pinyin and their corresponding Sino-Vietnamese meanings.

- **`CVDICT_Trung_Viet.csv`**
  A Chinese-Vietnamese dictionary dataset used for enriching the vocabulary lists with accurate translations and metadata.

## Overview

The vocabulary data in this project has been merged from multiple dictionary sources (`Thieu_Chuu_Dictionary.csv`, `hanviet.csv`, and the CVDICT dataset). It has been carefully cleaned to ensure proper alignment of Traditional and Simplified characters, as well as accurate Pinyin generation using the `pypinyin` library.
